Enhance the Crop Yield through Water Optimization

Designed an integrated control system using an Arduino microcontroller for automated environmental monitoring and control. The project utilized DHT11 temperature and humidity sensors, soil moisture sensors, a DC pump, and a GSM module, all connected through a relay. The system automates tasks based on sensor readings: if the temperature exceeds a set limit and soil moisture is low, the pump activates to irrigate plants. An LCD display provides real-time sensor readings for user monitoring. Additionally, the GSM module sends SMS alerts for critical events, such as high temperatures or low soil moisture, ensuring timely interventions. Sensor data is uploaded to a web server for remote monitoring and analysis, aiding in data-driven decision-making. This system enhances plant health through automated irrigation and offers users easy interaction and timely notifications.
